### Project 2: Federal R&D Budget — Tidy Data Analysis
**Folder:** [`TidyData-Project/`](./TidyData-Project/)

A Jupyter Notebook project that applies **tidy data principles** (Wickham, 2014) to clean and analyze U.S. Federal R&D spending from 1976–2017. The raw dataset has messy wide-format column headers that encode year and GDP together. Using `pd.melt()`, `str.split()`, and `str.replace()`, the data is reshaped into a clean, analysis-ready format. The project includes a pivot table, two visualizations, and thorough markdown documentation throughout.

**Key skills:** Python, Pandas, Matplotlib, Seaborn, data wrangling, tidy data principles, EDA

**How it complements the portfolio:** While the Streamlit app focuses on building interactive tools for end users, the Tidy Data project demonstrates the foundational data engineering work that any analysis depends on — knowing how to recognize and fix untidy data structures is a core skill for any data scientist. Together, these two projects show both the front-end (interactive apps) and back-end (data cleaning and transformation) sides of data science work.
